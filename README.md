# Shop Invest UI

## Getting Started

Clone the git repository on your local machine.

### Prerequisites

To can develop we need to install on the local machine:
1. node version >= 16
2. npm

## Environment

```bash
copy .env.example .env
```

## Running project

```bash
npm install
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
